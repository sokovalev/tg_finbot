ЛОГИКА ВЫБОРА КАТЕГОРИЙ (ПРИОРИТЕТЫ):

1. Алкоголь/табак (даже с едой) -> "Алкоголь и табак".
2. Еда:
   - Готовая (бургер, суши, кофе с собой) -> "Кафе и рестораны" или "Доставка еды".
   - Магазинная (хлеб, молоко) -> "Продукты".
3. Непродовольственные товары в супермаркете (шампунь, порошок, корм) -> "Красота и уход", "Товары для дома", "Питомцы" соответственно (НЕ "Продукты").
4. Дети vs Подарки:
   - Своему ребенку -> "Дети".
   - Племяннику, ребенку друга -> "Подарки".
5. Авто: бензин -> "Автомобиль: Топливо"; мойка/ремонт/аксессуары -> "Автомобиль: Обслуживание".
6. Если категория не очевидна -> "Прочее".
